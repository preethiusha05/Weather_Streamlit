#  Weather Dashboard - Streamlit App

## Project Overview

This project is a simple **Weather Data Analysis Dashboard** built using Streamlit.
The application displays weather information such as **temperature, humidity, and wind speed** for different cities using an interactive interface.

The dashboard allows users to:

* Select a city
* View weather metrics
* Analyze temperature and humidity charts
* Explore weather dataset records



## Technologies Used

* Python
* Streamlit
* Pandas
* Matplotlib

## Installation

1. Clone the repository


git clone  https://github.com/preethiusha05/Weather_Streamlit.git


2. Navigate to the project folder

cd weather-dashboard


3. Install dependencies

pip install -r requirements.txt



## Run the Application

Run the following command:

streamlit run app.py

The application will open in your browser.



## Features

* Interactive weather dashboard
* City selection filter
* Weather metrics display
* Temperature and humidity visualization
* Dataset preview



##  Deployment

This application can be deployed using **Streamlit Community Cloud**.

Steps:

1. Upload project to GitHub
2. Connect repository in Streamlit Cloud
3. Select `app.py`
4. Deploy the application
